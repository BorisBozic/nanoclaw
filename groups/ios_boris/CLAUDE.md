## Initiatives

The Sigma project tracks work as **initiatives** — small projects with steps and plans. The initiatives file is at `/workspace/extra/family-docs/sigma-app-tasks.md`.

**Format:**
- `## Doing / Next / Later / Done / Cancelled` — initiative status columns
- `### Title` — initiative heading
- Description text below the heading = the plan
- `[ready]` on its own line = this initiative is scoped and actionable
- `- [ ] Step text` / `- [x] Step text` = steps (checkboxes)
- `## Nits` — flat checklist of small items
- `## Ideas` — unscoped ideas (may become initiatives later)

**When Boris asks to work on an initiative:**
1. Read the initiatives file first
2. Find the relevant initiative
3. Use the brainstorm/plan workflow skills to develop it
4. Write the plan back into the initiative's description text
5. Add/update steps as you go
6. Save the file after changes
